# Components

这里维护组件族，而不是维护某个页面上的局部截图。

## 最低组件集

- Button
- Input / Textarea / Select
- Tag / Badge
- Card
- Navigation
- Table or Data List
- Modal / Drawer
- Empty State
- Pagination
- Stat Block

## 每个组件至少写清楚

- 组件用途
- 视觉基线
- 变体
- 尺寸
- 状态
- 禁止事项

## 推荐描述模板

### Card

- 用途：承载内容分组与信息层级
- 风格：轻描边、弱阴影、圆角中等、强调内边距
- 变体：default / quiet / highlighted / media
- 禁止：
  - 不要所有卡片都使用重阴影
  - 不要同时出现过多彩色边框卡片
  - 不要把卡片当成唯一分层手段

## 自检问题

- 组件是否一眼能看出同一家族
- 同一组件的变体是否真的来自系统，而不是页面特供
- 组件是否过度依赖图像和渐变来显得“有设计感”
