# Colors Foundation

这里记录视觉系统的色彩规律，而不是单纯罗列一堆颜色值。

## 必填内容

- 主色家族与辅助色家族
- 中性色阶
- 状态色
- 背景层级
- 文本对比层级

## 推荐格式

### Color Mood

- 关键词：克制、清透、可信、偏编辑感
- 避免：高饱和紫色、过重霓虹感、廉价渐变

### Palette

- Brand: `#0F766E`, `#115E59`, `#14B8A6`
- Neutral: `#0F172A`, `#334155`, `#64748B`, `#CBD5E1`, `#F8FAFC`
- Accent: `#C084FC`, `#FDBA74`
- Success / Warning / Danger: 按状态补齐

### Usage Rules

- 大面积背景优先中性色，不用高饱和品牌色铺满
- CTA 只允许使用主强调色，不允许每个区域都抢主按钮
- 同一页面的高对比色数量尽量不超过 2 组

## 自检问题

- 不看 logo，光看配色是否也能感到是同一套语言
- 主色是否被滥用到所有区域
- 背景、边框、文本是否有明确层级，而不是全靠纯白加阴影
