# Spacing / Radius / Shadow / Motion Foundation

这一层决定页面是否有节奏感和高级感，往往比单个颜色更影响审美判断。

## 必填内容

- 间距系统
- 圆角策略
- 阴影层级
- 描边策略
- 动效原则

## 推荐格式

### Spacing

- 4, 8, 12, 16, 24, 32, 48, 64, 96
- 组件内部间距和 section 间距分开定义

### Radius

- `radius/small`: 10
- `radius/medium`: 16
- `radius/large`: 24
- `radius/pill`: 999

### Shadow / Border

- 默认以轻描边加弱阴影为主
- 卡片层级优先通过背景与边框差异拉开，不靠重投影
- 阴影不超过 3 个层级

### Motion

- 动效以进入、层级切换、列表错峰出现为主
- 不做无意义 hover 花活
- 同一页面只保留 1 到 2 种主要动画语法

## 自检问题

- 页面节奏是否只靠“大空白”，还是有层级化的间距逻辑
- 圆角是否所有组件都一样，导致没有性格
- 阴影是否廉价且泛滥
