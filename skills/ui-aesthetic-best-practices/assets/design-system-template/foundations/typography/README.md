# Typography Foundation

这里记录排版节奏，而不是只记录字号。

## 必填内容

- 标题层级
- 正文层级
- 字重策略
- 行高策略
- 字距策略
- 中英文混排注意项

## 推荐格式

### Type Mood

- 关键词：编辑感、利落、稀疏留白、非营销海报感
- 避免：每级标题都过粗、正文行高过挤、说明文字过灰过小

### Scale

- Display: 56/64, weight 600
- H1: 40/48, weight 600
- H2: 32/40, weight 600
- H3: 24/32, weight 600
- Body L: 18/30, weight 400
- Body M: 16/28, weight 400
- Body S: 14/22, weight 400
- Caption: 12/18, weight 500

### Rules

- 一个页面最多一个 display 级标题区域
- H1 到 H3 之间要有明显张力，不要只差 2px
- 描述性文本优先靠行高和列宽获得阅读感，不靠一味变浅
- 数字、金额、数据指标允许使用更紧凑的字距

## 自检问题

- 不看颜色，只看文本排布是否仍然有层次
- 页面是否出现了过多“看起来都差不多”的标题级别
- 正文是否能连续阅读而不觉得累
