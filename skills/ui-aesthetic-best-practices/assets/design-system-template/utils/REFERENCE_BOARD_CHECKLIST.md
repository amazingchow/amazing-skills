# Reference Board Checklist

在把参考网页转进 Figma 之后，用这份清单筛一次。

## 来源质量

- 是否至少有 3 个参考源
- 每个参考源是否只承担 1 个主要维度
- 是否避免了风格过近、导致抽象结果过窄

## 去品牌化

- 是否删除了 logo
- 是否删除了品牌 slogan
- 是否删除了强识别插画
- 是否删除了专有图形和 icon 语言

## 可抽象内容

- 是否已经提炼色彩家族
- 是否已经提炼字号与排版节奏
- 是否已经归纳常见组件
- 是否已经记录 section 级 pattern

## 风险判断

- 是否仍然一眼就能认出原站
- 是否过度依赖单一参考源
- 是否已经能明确说出“我们要保留什么，不保留什么”
